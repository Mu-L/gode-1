title: Overview
description: TypeScript scripting runtime for Godot projects, with native platform support, the npm ecosystem, and reproducible exports.
---

Gode adds JavaScript/TypeScript language support to Godot. Write Godot node scripts in modern TypeScript, import Godot classes from the `godot` module, and use the powerful `npm` ecosystem.

## Platform support

Gode targets native Godot projects and supports the following platforms.

| Platform | Supported | Minimum version |
| --- | --- | --- |
| Windows | Yes | Windows 10 |
| Android | Yes | Android 9 |
| macOS | Yes | macOS 10.15 |
| iOS | Yes | iOS 16 |
| Linux | Yes | Ubuntu 22 |
