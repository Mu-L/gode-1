Gode 为 Godot 增加了 JavaScript/TypeScript 语言支持。使用现代 TypeScript 编写节点脚本，从 `godot` 模块导入 Godot 类，无缝接入强大的 `npm` 生态。

## 平台支持

Gode 面向原生 Godot 项目，支持所有主流系统。

| 平台 | 支持 | 最低版本 |
| --- | --- | --- |
| Windows | ✅ | Windows 10 |
| Android | ✅ | Android 9 |
| macOS | ✅ | macOS 10.15 |
| iOS | ✅ | iOS 16 |
| Linux | ✅ | Ubuntu 22 |
