Gode 为 Godot 增加了 JavaScript/TypeScript 语言支持。使用现代 TypeScript 编写节点脚本，从 `godot` 模块导入 Godot 类，无缝接入强大的 `npm` 生态。

## 平台支持

Gode 面向原生 Godot 项目，支持以下平台。

| 平台 | 支持 | 最低版本 |
| --- | --- | --- |
| Windows | 是 | Windows 10 |
| Android | 是 | Android 9 |
| macOS | 是 | macOS 10.15 |
| iOS | 是 | iOS 16 |
| Linux | 是 | Ubuntu 22 |
